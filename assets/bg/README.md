# Fondos de ruta / combate

Coloca aquí las imágenes de fondo para cada ruta.
El nombre debe coincidir con el atributo `bg` de ROUTE_DATA en routes.js.

Archivos esperados:
  route-1.png       — Ruta 1
  route-2.png       — Ruta 2
  bosque-verde.png  — Bosque Verde

Tamaño recomendado: 480x280px (proporción del campo de combate)
Formatos soportados: png, jpg, webp
