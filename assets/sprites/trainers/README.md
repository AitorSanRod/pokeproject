# Sprites de entrenadores

Coloca aquí las imágenes de los entrenadores.
El nombre debe coincidir con el atributo `img` de cada entrenador en routes.js.

Archivos esperados:
  youngster.png    — Youngster Joey
  lass.png         — Lass Amy
  bug-catcher.png  — Bug Catcher

Tamaño recomendado: 32x32px o 48x48px, PNG con transparencia.
